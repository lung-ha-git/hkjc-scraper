## 功能 Specification

### 1. 賽事列表 (左側/頂部Tab)
- R1 - R10 (或R11)
- 顯示: 場次、途程、總獎金
- Click to select race

### 2. Race Card 顯示
- 馬匹編號、馬名、騎師、檔位
- 評分、體重、配備
- 6次近績

### 3. Prediction Panel
- **排名**: 1st, 2nd, 3rd, 4th
- **預測分數**: weighted score
- **勝出概率**: Monte Carlo模擬

### 4. 因子權重調整 (右側/底部)
| 因子 | 預設權重 | 範圍 |
|------|----------|------|
| hj_win_rate | 10 | 0-20 |
| career_place_rate | 5 | 0-10 |
| jockey_win_rate | 3 | 0-10 |
| trainer_win_rate | 2 | 0-10 |
| dist_win_rate | 2 | 0-10 |
| recent3_avg_rank | 2 | 0-5 |
| current_rating | 1 | 0-5 |
| dist_wins | 1 | 0-5 |
| jt_win_rate | 1 | 0-5 |
| draw | -1 | -5 to 0 |
| randomness | 3 | 0-10 |

### 5. API Endpoints
```
GET  /api/races              # 賽事列表
GET  /api/races/:id         # 單場賽事+馬匹
POST /api/predict           # 預測 (body: race_id, weights)
GET  /api/predictions/:date # 某日預測
```

### 6. Data Sources
- MongoDB: races, horses, jockeys, trainers
- Real-time scrape from HKJC (optional)

---

## Tech Stack
- **Frontend**: React + Tailwind CSS
- **Backend**: FastAPI / Express
- **Database**: MongoDB (existing)
- **Deployment**: Vercel / Railway

---

## Implementation Phases

| Phase | Description |
|-------|-------------|
| Phase 1 | 基本Race Card UI (靜態數據) |
| Phase 2 | 連接MongoDB, 顯示真實數據 |
| Phase 3 | Weighted Prediction Algorithm |
| Phase 4 | 因子權重調整UI |
| Phase 5 | Mobile Responsive |
| Phase 6 | Real-time sync from HKJC |

4.3 模型訓練流程

``` 1. 數據準備 ├── 收集歷史賽事數據 (2020-2026) ├── 清洗缺失值與異常值 └── 分割訓練/驗證/測試集 (70/15/15)

  1. 特徵工程 ├── 數值特徵標準化 ├── 類別特徵編碼 (One-hot/Label) └── 特徵選擇 (Feature Importance)

  2. 模型訓練 ├── XGBoost 分類器 ├── Random Forest 分類器 └── MLP 神經網絡

  3. 模型評估 ├── Accuracy, Precision, Recall ├── AUC-ROC 曲線 ├── Log Loss └── 收益回測 (Backtesting)

  4. 持續學習 ├── 每週收集新數據 ├── 增量訓練 (Incremental Training) └── 模型版本管理 ```

5. 投注優化算法

5.1 投注類型與回報

| 投注類型 | 描述 | 典型回報率 | |---------|------|-----------| | Win | 單選頭馬 | 1.0-30x | | Place | 單選前三 | 1.0-5x | | Quinella | 順序不拘選2隻 | 5-50x | | Quinella Place | 任2隻跑入前三 | 2-20x | | Tierce | 順序正確選3隻 | 50-500x | | Quartet | 順序正確選4隻 | 100-2000x |

5.2 投注優化目標函數

``` 最大化預期收益:

Maximize: Σ (P(win) × 赔率 - 1) × 投注金額

約束條件: - 總投注金額 ≤ 資金上限 - 單場投注 ≤ 總資金的 X% - 風險價值 (VaR) ≤ 可承受損失 ```

5.3 投注策略

策略一：高信心投注 (High Confidence)

  • 條件：模型預測勝出概率 > 40%
  • 投注：根據 Kelly Criterion 計算金額
  • 預期回報：中至高

策略二：價值投注 (Value Betting)

  • 條件：模型概率 × 赔率 > 1.1 (期望值 > 10%)
  • 投注：固定比例或遞增
  • 預期回報：中

策略三： quinella 組合

  • 條件：兩匹馬概率互補 (一隻高 + 一隻中)
  • 投注：覆蓋多個組合
  • 預期回報：中至高，較穩定

5.4 Kelly Criterion 應用

``` f* = (bp - q) / b

其中: f* = 應投注比例 b = 赔率 - 1 p = 獲勝概率 q = 失敗概率 = 1 - p

建議調整: - 半 Kelly: f* / 2 (較保守) - Quarter Kelly: f* / 4 (極保守) ```

9.3 資金風險

  • 單場投注不超過總資金的 5%
  • 單日投注不超過總資金的 15%
  • 設置止損線 (-20% 總資金)

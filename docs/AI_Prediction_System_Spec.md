## 3. 預測因子 (Prediction Features)

### 3.1 馬匹因子 (Horse Factors)

| 因子 | 計算方式 | 權重評估 |
|------|---------|---------|
| 近期表現 | 最近5-10場平均名次 | 高 |
| 距離適應性 | 同距離歷史勝出率 | 高 |
| 跑道適應性 | 草地/泥地表現 | 高 |
| 評分優勢 | 對手平均評分 vs 自己評分 | 中 |
| 體重變化 | 近期體重增減 | 中 |
| 休息時間 | 上一場至今相隔日數 | 中 |
| 檔位優劣 | 檔位歷史表現 | 低 |
| 血統潛力 | 父系同距離表現 | 低 |

### 3.2 騎師因子 (Jockey Factors)

| 因子 | 計算方式 | 權重評估 |
|------|---------|---------|
| 騎師勝出率 | 季內勝出/總騎乘 | 高 |
| 合作練馬師 | 與練馬師合作勝率 | 中 |
| 近期狀態 | 最近10場表現 | 中 |
| 對手了解 | 對手騎師勝率對比 | 低 |

### 3.3 練馬師因子 (Trainer Factors)

| 因子 | 計算方式 | 權重評估 |
|------|---------|---------|
| 練馬師勝出率 | 季內勝出/總參賽 | 高 |
| 馬匹準備度 | 操練表現/試閘成績 | 中 |
| 季內狀態 | 近期頭馬數量 | 中 |

### 3.4 賽事因子 (Race Factors)

| 因子 | 計算方式 | 權重評估 |
|------|---------|---------|
| 班次差距 | 高班馬對低班馬優勢 | 高 |
| 距離適合度 | 馬匹最佳距離 vs 賽事距離 | 高 |
| 負磅影響 | 負磅對速度的影響 | 中 |
| 跑道狀況 | 馬匹擅長跑道 vs 實際跑道 | 中 |
| 檔位影響 | 內檔/外檔歷史表現 | 中 |

### 4.2 目標變量 (Target Variables)

| 預測目標 | 類型 | 描述 |
|---------|------|------|
| Win | Binary | 馬匹是否獲得第1名 |
| Place | Binary | 馬匹是否獲得前3名 |
| Quinella | Binary | 選擇的 quinella 組合是否中獎 |
| Exacta | Binary | 選擇的 exacta 順序是否正確 |
| 排名 | Regression | 預測馬匹的具體名次 |

### 4.3 模型訓練流程

```
1. 數據準備
   ├── 收集歷史賽事數據 (2020-2026)
   ├── 清洗缺失值與異常值
   └── 分割訓練/驗證/測試集 (70/15/15)

2. 特徵工程
   ├── 數值特徵標準化
   ├── 類別特徵編碼 (One-hot/Label)
   └── 特徵選擇 (Feature Importance)

3. 模型訓練
   ├── XGBoost 分類器
   ├── Random Forest 分類器
   └── MLP 神經網絡

4. 模型評估
   ├── Accuracy, Precision, Recall
   ├── AUC-ROC 曲線
   ├── Log Loss
   └── 收益回測 (Backtesting)

5. 持續學習
   ├── 每週收集新數據
   ├── 增量訓練 (Incremental Training)
   └── 模型版本管理
```

## 5. 投注優化算法

### 5.1 投注類型與回報

| 投注類型 | 描述 | 典型回報率 |
|---------|------|-----------|
| Win | 單選頭馬 | 1.0-30x |
| Place | 單選前三 | 1.0-5x |
| Quinella | 順序不拘選2隻 | 5-50x |
| Quinella Place | 任2隻跑入前三 | 2-20x |
| Tierce | 順序正確選3隻 | 50-500x |
| Quartet | 順序正確選4隻 | 100-2000x |

### 5.2 投注優化目標函數

```
最大化預期收益:

Maximize: Σ (P(win) × 赔率 - 1) × 投注金額

約束條件:
- 總投注金額 ≤ 資金上限
- 單場投注 ≤ 總資金的 X%
- 風險價值 (VaR) ≤ 可承受損失
```

### 5.3 投注策略

#### 策略一：高信心投注 (High Confidence)
- 條件：模型預測勝出概率 > 40%
- 投注：根據 Kelly Criterion 計算金額
- 預期回報：中至高

#### 策略二：價值投注 (Value Betting)
- 條件：模型概率 × 赔率 > 1.1 (期望值 > 10%)
- 投注：固定比例或遞增
- 預期回報：中

#### 策略三： quinella 組合
- 條件：兩匹馬概率互補 (一隻高 + 一隻中)
- 投注：覆蓋多個組合
- 預期回報：中至高，較穩定

### 5.4 Kelly Criterion 應用

```
f* = (bp - q) / b

其中:
f* = 應投注比例
b = 赔率 - 1
p = 獲勝概率
q = 失敗概率 = 1 - p

建議調整:
- 半 Kelly: f* / 2 (較保守)
-  Quarter Kelly: f* / 4 (極保守)
```

## 9. 風險與限制

### 9.1 技術風險

| 風險 | 影響 | 緩解措施 |
|------|------|---------|
| 數據質量 | 預測不準 | 數據清洗與驗證 |
| 模型過擬合 | 預測偏差 | 交叉驗證 |
| 即時赔率變化 | 投注失效 | 實時監控與自動化 |

### 9.2 業務風險

| 風險 | 影響 | 緩解措施 |
|------|------|---------|
| 馬匹受傷 | 退出比賽 | 及時更新數據 |
| 騎師更換 | 影響表現 | 預留替代方案 |
| 市場波動 | 赔率變化 | 設定合理的投注閾值 |

### 9.3 資金風險

- 單場投注不超過總資金的 5%
- 單日投注不超過總資金的 15%
- 設置止損線 (-20% 總資金)
